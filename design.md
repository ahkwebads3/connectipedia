# Audience Analysis Tool - Design Style Guide

## Design Philosophy
- **Modern Minimalism**: Clean, uncluttered interface with purposeful design elements
- **Professional Elegance**: Sophisticated aesthetic that conveys expertise and reliability
- **User-Centric Approach**: Intuitive navigation and clear information hierarchy
- **Arabic Typography**: Optimized for Arabic script with proper font selection

## Color Palette
- **Primary Colors**:
  - Pure White (#FFFFFF) - Main background
  - Light Gray (#F8F9FA) - Section backgrounds
  - Charcoal Gray (#2C3E50) - Primary text
  - Professional Blue (#1e3a8a) - Interactive elements and highlights (Amr Galal brand color)
- **Secondary Colors**:
  - Success Green (#27AE60) - Progress indicators
  - Warning Orange (#F39C12) - Attention elements
  - Soft Gray (#95A5A6) - Secondary text
  - Deep Navy (#1e40af) - Professional accent color

## Typography
- **Primary Font**: 'Tajawal' - Modern Arabic font with clean lines
- **Secondary Font**: 'Cairo' - Alternative Arabic font for headings
- **Font Hierarchy**:
  - H1: 2.5rem, bold - Main headings
  - H2: 2rem, semibold - Section headings
  - H3: 1.5rem, medium - Subsection headings
  - Body: 1rem, regular - Main content
  - Small: 0.875rem - Helper text

## Visual Language
- **Geometric Shapes**: Subtle use of circles and rounded rectangles
- **Soft Shadows**: Minimal depth with light drop shadows
- **Consistent Spacing**: 8px grid system for uniform layout
- **Rounded Corners**: 8px border radius for modern feel
- **Icon Style**: Outline icons with consistent stroke width

## Layout Principles
- **Mobile-First**: Responsive design starting from 320px width
- **Grid System**: 12-column flexible grid layout
- **Whitespace**: Generous spacing for visual breathing room
- **Content Hierarchy**: Clear visual hierarchy with size and color
- **Centered Alignment**: Content centered with max-width constraints

## Interactive Elements
- **Buttons**: 
  - Primary: Blue background, white text, rounded corners
  - Secondary: White background, blue border and text
  - Hover: Subtle scale animation and color transition
- **Form Elements**:
  - Clean input fields with subtle borders
  - Focus states with blue accent color
  - Clear labels and placeholder text
- **Cards**:
  - White background with soft shadows
  - Rounded corners and subtle hover effects
  - Consistent padding and margins

## Visual Effects
- **Animations**: Smooth transitions using Anime.js
- **Background**: Subtle gradient overlay with shader effects
- **Progress Indicators**: Animated progress bars with completion states
- **Hover Effects**: Gentle scale and shadow transitions
- **Loading States**: Skeleton screens and progress animations

## Accessibility
- **Contrast Ratios**: Minimum 4.5:1 for all text
- **Focus Indicators**: Clear keyboard navigation
- **Screen Reader**: Proper ARIA labels and semantic HTML
- **Touch Targets**: Minimum 44px for mobile interactions
- **Language**: RTL support for Arabic interface
